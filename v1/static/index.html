Kcore Database API

To upload data

Please refer to the distributed submit script.

To download data

Get networks:
http://134.74.76.229:8080/getnetworks?cut=30

  • "cut" is a data cutoff option in days, default 30 days.
  • Returned networks are calculated within cutoff days.
  • Statistics such as last visited time and clicks are those within cutoff days

Get influencers of a network:
http://134.74.76.229:8080/get?keyword=global%20warming

  • If you know network name, use this to get the network information.

Get queued keywords in recent to old order:
http://134.74.76.229:8080/queued?cut=30

  • "cut" is a data cutoff option in days, default 30 days.
  • This is a convenience function that returns a simple list for dataminer.
  • To get full statistics, use getnetworks.
