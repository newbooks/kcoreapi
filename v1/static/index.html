Kcore Database API

To upload data

Please refer to the distributed submit script.

To download data

Get networks:
http://134.74.76.229:8080/getnetworks?cut=30
"cut" is a data cutoff option in days, default 30 days.

Get influencers of a network:
http://134.74.76.229:8080/get?keyword=global%20warming

Get history of a handler in a network:
http://134.74.76.229:8080/gethistory?keyword=global%20warming&handler=@evanderburg

Get queued keywords in recent to old order:
http://134.74.76.229:8080/queued?cut=30
"cut" is a data cutoff option in days, default 30 days.

Get hottest search keywords in high to low order:
http://134.74.76.229:8080/hottest?cut=30&limit=10
"cut" is a data cutoff option in days, default 30 days. "limit" is maximum number of items returned.
